# 链上生态

| **模块** | **功能描述**            | **链接**                                                               |
| ------ | ------------------- | -------------------------------------------------------------------- |

| 跨链桥    | 实现 JU 代币与其他区块链的跨链转移 | [bridge-testnet.juchain.org](https://bridge-testnet.juchain.org)     |
